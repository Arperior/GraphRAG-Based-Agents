# ======================= GraphRAG Extract Graph Prompt =======================

Goal:
Given a text document, identify all entities and relationships among them.
Output MUST be in valid JSON.

Steps:
1. Extract all entities in the text.
2. For each entity, extract:
   - name
   - type (PERSON, ORGANIZATION, GEO, EVENT, DATE, etc.)
   - description

3. For each pair of entities, extract relationships, with:
   - source
   - target
   - relation (UPPERCASE, underscore style, e.g., CAPITAL_OF)
   - evidence (the text supporting the relation)
   - confidence (float 0.0â€“1.0)

Return ONLY valid JSON with the format:

  "relations": [
    {"source": "Delhi", "target": "India", "relation": "CAPITAL_OF", "evidence": "Delhi is the capital of India.", "confidence": 1.0}

}
