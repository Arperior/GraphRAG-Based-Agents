1. Make a new folder 
2. Have python 3.12.5 installed
3. run : pip install uv
4. Create venv using : uv venv --python 3.12
5. run : .venv\Scripts\Activate.ps1
6. If security error run : Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
7. Once it shows name of venv like (venv) then :
    - IN CASE OF PIP ERROR: python -m ensurepip --upgrade
    - uv pip install graphrag openai spacy nltk python-dotenv poethepoet
    - python -m spacy download en_core_web_md
8. Once done, execute : " graphrag init --root . "
9. Install JDK 21 :- https://docs.oracle.com/en/java/javase/24/install/installation-jdk-microsoft-windows-platforms.html#GUID-A7E27B90-A28D-4237-9383-A58B416071CA
10. Download Neo4j :- https://neo4j.com/docs/operations-manual/current/installation/windows/
11. Install CE(not as service) as listed in the installation guide.
12.Download APOC JAR from GitHub (matching Neo4j version).
13.Copy APOC JAR to ...\Neo4jDesktop2\Data\dbmss\<dbms-id>\plugins\
14.Edit neo4j.conf → add:
    -dbms.security.procedures.unrestricted=apoc.*,gds.*
    -dbms.security.procedures.allowlist=apoc.*,gds.*
    -apoc.import.file.enabled=true
    -apoc.export.file.enabled=true
    -apoc.trigger.enabled=true
    -dbms.default_database=neo4j
15.Restart Neo4j Desktop.
16.Open Neo4j Browser → Run: RETURN apoc.version();
17.Download Mistral model (quantized GGUF version):
    -Go to: https://huggingface.co/TheBloke/Mistral-7B-Instruct-v0.1-GGUF
    -Download file: mistral-7b-instruct-v0.1.Q4_K_M.gguf
    -Create folder: D:\models\Mistral-7B-Instruct-v0.1-GGUF\
    -Move the model file there.
18.Create .env in project root with:
    -NEO4J_URI=bolt://localhost:7687
    -NEO4J_USER=neo4j
    -NEO4J_PASSWORD=your_password
    -GEMINI_API_KEY=your_gemini_api_key
    -LOCAL_LLM_MODEL=D:\models\Mistral-7B-Instruct-v0.1-GGUF
    -LOCAL_LLM_FILE=mistral-7b-instruct-v0.1.Q4_K_M.gguf
    -LEIDEN_RESOLUTION=1.0
    -RETRIEVAL_SEARCH_LIMIT=10
    -NEO4J_QUERY_LIMIT=100
19.Test APOC in Python: python -m pipeline.neo4j_client
20. Run mistral test: testing.py
21.Run app: python app.py
